title: "Utilities"
---

Neon's utility commands make running your server easier and more fun.

## Popular Utilities

- **afk** — Set yourself as AFK.
- **avatar** — View a user's avatar.
- **banner** — Show a user's banner.
- **enlarge** — Enlarge server emojis.
- **firstmessage** — Find the first message in a channel.
- **help** — Get info on all commands.
- **image** — Search and display images.
- **inrole** — List users with a specific role.
- **invite** — Get the bot's invite link.
- **membercount** — Show how many members are in the server.
- **nasa** — NASA image of the day.
- **ping** — Check the bot's latency.
- **reminder** — Set a reminder.
- **roleinfo** — Info about a specific role.
- **roles** — List all server roles.
- **serverbanner** — Show the server's banner.
- **servericon** — Show the server's icon.
- **serverinfo** — Info about the server.
- **serversplash** — Show the server's splash image.
- **snipe** — Show the last deleted message.
- **spotify** — Share your Spotify status.
- **support** — Get a link to the support server.
- **tiktok** — Download TikTok videos.
- **timezone** — Show time in a specific timezone.
- **uptime** — How long Neon has been online.
- **urban** — Urban Dictionary definitions.
- **userinfo** — Info about a user.
- **userroles** — List a user's roles.
- **weather** — Weather info for a city.
- **wolfram** — Wolfram Alpha queries.
- **youtubedown** — Download YouTube videos.

Type `n!help` or any command for more info.
