# Neon Bot, Documentation

Thank you for using neon bot, learn more about the bot, its features and how to get started here.

## Getting Started

Neon is a bot that uses mainly prefix commands, that means you type the prefix followed by the command you wish to run. 

<Info>
  The default prefix used by neon is `n!`
</Info>

/

<Info>
  After adding neon to your server, the default prefix will be `n!` 

  You can also mention the bot to see its prefix.

  If you'd like to change it, use`n!prefix <new prefix>`
</Info>
