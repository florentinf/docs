# Neon Bot, Documentation

Thank you for using neon bot, learn more about the bot, its features and how to get started here.

## Information

Neon is a bot that uses mainly prefix commands, that means you type the prefix followed by the command you wish to run.

<Info>
  Mention the bot to see the prefix of the bot on a server. The default prefix used by neon is `n!`

  ![SCR_2025-09-17_14-33-08_f145023@2x.png](/images/SCR_2025-09-17_14-33-08_f145023@2x.png)
</Info>

<Note>
  Change the prefix of the bot using `n!prefix < new prefix >`
</Note>
