## Information

Neon is a bot that mainly uses prefix commands. This means you type the prefix followed by the command you want to run.

<Info>
  Mention the bot to see the prefix of the bot on a server. The default prefix used by neon is `n!`

  <img
    src="/images/SCR_2025-09-17_14-33-08_f145023@2x.png"
    alt="SCR_2025-09-17_14-33-08_f145023@2x.png"
    style={{ width:"94%" }}
  />
</Info>

<Note>
  Change the bot's prefix using `n!prefix <new prefix>`
</Note>
